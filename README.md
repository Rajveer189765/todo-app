# Todo App

A full-stack task management application built with the MERN stack.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Create Task
- Get All Tasks
- Get Single Task
- Update Task
- Delete Task

## API Endpoints

POST   /api/auth/register
POST   /api/auth/login

## Project Status

Backend authentication completed.
Task management and frontend are under development.
